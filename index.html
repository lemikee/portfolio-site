About Me

As a Private Equity Analyst turned full-stack software engineer, I enjoy using React/Redux, Javascript, Ruby on Rails, PostgreSQL, HTML, and CSS to build responsive projects that scale. I'm always learning new technologies and most recently learned the MERN stack and lead a team to build an innovative job matching app, blindr, all in one week.

As an Applied Mathematics graduate from UC Berkeley, I was driven to solve complex problems by writing rigorous mathematical proofs. I enjoyed that the challenge of having to continually learn new tools to formulate efficient solutions. As an undergraduate, I washed dishes in the dining halls to pay tuition and eventually progressed to starting my own IT support business. Having built the business from the ground up, I wanted to learn from others how to scale and run businesses at the highest levels. As a Private Equity Analyst, I founded and led a team to drive over $50M+ in aggregate deal flow, with the mission to help amazing companies sustainable scale and grow. Now, as a software engineer, I'm able to leverage my prior experience to help deliver efficient and powerful solutions through code.

Growing up, I didn't really have the opportunity to try new foods. Now as an adult, I love trying new foods all the time and one of the main tools I use to find new foods is Yelp. As a passion project, I build a full-stack clone of Yelp where users are able to search for new businesses, read reviews by other users, write/edit their own reviews, view pictures, and login/sign up just like on Yelp! I welcome you to checkout bytes, and my other projects below!

bytes

bytes is a pixel-perfect Yelp clone built on React/Redux, Ruby on Rails, PostgreSQL, HTML, and CSS. Users are able to search for new businesses, read reviews by other users, write/edit their own reviews, view pictures, and login/sign up just like on Yelp!

blindr

blindr is a new and efficient job matching app where the only thing that matters is your skills - and nothing else. Setup your profile with your skills and quickly swipe to match with jobs sourced based off of your profile. Once you're matched, you're able to chat with employers directly in the app! Built using the MERN (MongoDB, Express, React, NodeJS) stack.

Pokemon Battle

Pokemon Battle is a javascript game that recreates the turn-based strategy battling functionality that made the Pokemon series so famous. Built using vanilla Javascript, Vanilla DOM, and HTML/CSS.
